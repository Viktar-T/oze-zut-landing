# 2.2 Solar-Wind Hybrid Station 

Solar/Wind hybrid energy generation station combining both solar and wind energy capabilities for research and education.

**Type:** Hybrid Energy System  
**Configuration:** Solar-Wind Hybrid Station

## Data Acquisition System

### Current Situation

A PC is already directly connected to the hybrid station system. This PC gathers and stores data using manufacturer-provided software. The system manual indicates optional LAN integration for network-based data transfer.

### Recommendations

- Connect the PC directly to the local LAN
- Configure either file/data sharing, or direct streaming from the hybrid controller if supported
